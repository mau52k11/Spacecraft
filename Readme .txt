Spacecraft 

Lo que se pretende hacer es usar hilos para la realización de un pequeño videojuego.  Este será un shotter en tercera persona, el objetivo de este videojuego es destruir a los distintos tipos de enemigos que le aparecerán en el espacio y destruir al jefe final para poder escapar de dicha galaxia, estos podrán ser eliminados con misiles que disparará la nave, este no tendrá un limite en el número de veces que podrá disparar.
Para que este juego sea entretenido, se le aumentará la velocidad a los enemigos cada vez que el usuario mate a 100 enemigos, cuando solo quede un enemigo, el jefe tendrá aparición en dicho nivel, este jefe es mas grande a comparación de los demás enemigos.

Comandos:

Con las teclas de navegación te desplazas a la posición que uno elija.
con la tecla space, se dispara un misil que explotará al contacto con los enemigos.
